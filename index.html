Piskel-clone is a clone of free online animated sprite editor.

Exsample

Implemented functionality

  • Tools: pen, eraser, color picker, vertical mirror, rectangle, lighten
  • Preview: Small animation preview and Full screen mode
  • Frame management: add, delete, clone and reorder a frame via Drag and Drop
  • Keyboard shortcuts for tools and frame management
  • Show cursor coordinates
  • Save user session in localstorage
  • Full compatibility with original export format (.piskel)
